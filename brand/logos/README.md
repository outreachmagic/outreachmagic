# Logo files

## Naming

```
logo-icon-{variant}.svg       # Icon mark only (no text)
logo-horizontal-{variant}.svg   # Full logo with text, horizontal layout
```

`dark` / `black` = dark ink on light backgrounds. `white` = light ink on dark backgrounds.

## File reference

| File | Type | Use on | Description |
|------|------|--------|-------------|
| `logo-icon-black.svg` | SVG icon | Light backgrounds | Icon mark in `#222` |
| `logo-icon-white.svg` | SVG icon | Dark backgrounds | Icon mark in `#fff` |
| `logo-horizontal-dark.svg` | SVG full logo | Light backgrounds | Full "OutreachMagic" wordmark, dark |
| `logo-horizontal-white.svg` | SVG full logo | Dark backgrounds | Full "OutreachMagic" wordmark, white |
